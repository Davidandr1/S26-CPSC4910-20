Product Name: Good Driver Incentive Program
Product Description:
A web application for trucking companies to reward safer, better driving. Sponsors award and remove points based on driver performance, and drivers redeem those points for items in a sponsor-managed catalog. The system tracks point activity and purchases so the service can be sold and administered at scale.
Demo Notes:
In this sprint we worked on updating the UI to give users more information depending on role, worked on the API pulls, gained the ability to search through ebay api. We have added a leaderboard and the ability to adjust points. Quick apply events for point adjustments and more detailed product information. We have also added a lot of error handling and edge case handling to make the application more robust.
