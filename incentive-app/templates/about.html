Product Name: Good Driver Incentive Program
Product Description:
A web application for trucking companies to reward safer, better driving. Sponsors award and remove points based on driver performance, and drivers redeem those points for items in a sponsor-managed catalog. The system tracks point activity and purchases so the service can be sold and administered at scale.
Demo Notes:
This demo includes the link from web page to the DB. You can create and log in to a Driver, Sponsor, or Admin account. Each has their own unique homepage. We will be linking the About page to the DB soon. New Improvements are further built out routes on admin and user creations by sponsors. A password reset has also been set up and is currently being debugged.
