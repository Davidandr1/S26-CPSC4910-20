Product Name: Good Driver Incentive Program
Product Description:
A web application for trucking companies to reward safer, better driving. Sponsors award and remove points based on driver performance, and drivers redeem those points for items in a sponsor-managed catalog. The system tracks point activity and purchases so the service can be sold and administered at scale. We also added a category filter to both the storefront and to the sponsors product list.
Demo Notes:
In this sprint we worked on improving the point control by adding scheduled events, which is currently not fully automated, but can be with scheduler permission. We also added a point cap feature, which allows the sponsor to set a max value a user can hold, else 3,000,000 points.
